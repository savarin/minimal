# minimal

Minimal examples for compiling and running WebAssembly.

- v1 - .wat to .wasm
- v2 - Rust to .wasm
- v3 - Rust/JavaScript interop via wasm
- v4 - full stack Rust via wasm

Please refer to respective directories for details.
